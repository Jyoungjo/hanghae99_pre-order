== 공통 사항

API에 관계없이 아래 사항을 지켜주셔야 합니다.

=== Domain

|===
| 환경 | domain

| 개발서버
| `localhost`

| 운영서버
| `AWS EC2 - t3.large`
|===


=== Header

|===
| name | 설명

| `Cookie`
| accessToken : Access Token을 담고 있습니다.

refreshToken : Refresh Token을 담고 있습니다.
|===
